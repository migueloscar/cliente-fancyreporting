FANCY REPORTING

Una librería que genera reportes de indicadores en archivos de Microsoft Word para proyectos .Net Framework tanto para entornos web como de escritorio.

Instalación

Descargar la librería y agregar al proyecto de .Net Framework.

también se deben de agregar las librerías desde Nuget:

  • NewtonSoft.json - Librería para manejar objetos Json.
  • DocumentFormat.OpenXml - Librería para la generación de archivos Microsoft Word con Open XML.

Resultados

El reporte en formato Microsoft Word es el siguiente:

Los reportes constan de tres secciones:

  • Una tabla que muestra la evolución de los indicadores.
  • Párrafos que describen los valores de la tabla.
  • Un gráfico lineal donde se aprecia la evolución de los indicadores.

Las secciones de los párrafos y el gráfico lineal se repiten por cada grupo de la tabla.

Se puede personalizar el reporte modificando textos, columnas, indicar que datos deben de tener descripción y cuales no, de similar forma para el gráfico lineal. Para mayor información visite la Documentación.
