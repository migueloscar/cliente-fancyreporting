Implementación de la librería FancyReporting

Descargar la librería de la pagina web FancyReporting.miguelaguirre.dev.
Agregar la librería como referencia.

Preparando la información que consume la librería para generar el reporte.

modelo: Parámetro del tipo enum TipoReporte, que recibe el tipo de reporte que se desea implementar, usar el namespace ReportingGenerator.
listaDatosJson: Parámetro del tipo string, esta cadena contiene la información de los registros que se mostraran en el reporte en un formato Json; Este archivo Json es un array de objetos que tiene los siguientes miembros:

  • detalle del tipo string, es el nombre del registro.
  • periodo del tipo date, el periodo en el que se obtuvo este registro.
  • valor del tipo double, valor que obtuvo este registro en el periodo indicado anteriormente.
